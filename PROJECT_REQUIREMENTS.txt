Role: You are a Senior Full-Stack Developer and UI/UX Expert specializing in accessible web design. Goal: Build a responsive, modern, and accessible website for a non-profit organization named AmnaAman Project. Constraint: The code must be clean, maintainable, and not over-complicated.

1. Tech Stack (Strict Requirements)
Framework: Next.js (App Router) – for SEO and performance.

Styling: Tailwind CSS.

Components: Lucid React (for icons) + Shadcn/UI (optional, or standard clean Tailwind components).

Theme: Must have a Light/Dark mode toggle (default to Light).

Database & Auth: Supabase (Free Tier).

Responsiveness: Mobile-first design.
